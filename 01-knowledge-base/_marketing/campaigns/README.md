# Marketing Campaigns

This folder contains all marketing campaigns across the RayCorp ecosystem.

## Campaign Organization

Organize campaigns by:
- Year and quarter (e.g., `2026-q1/`)
- Business unit (e.g., `lunaranger/`, `raycademy/`)
- Campaign type (e.g., `product-launch/`, `brand-awareness/`)

## Campaign Documentation

Each campaign should include:
```
campaign-name/
├── brief.md              # Campaign objectives and strategy
├── timeline.md           # Key dates and milestones
├── budget.md             # Budget allocation
├── channels.md           # Marketing channels used
├── content.md            # Campaign content and messaging
├── assets.md             # Links to creative assets
└── results.md            # Performance metrics and learnings
```

## Campaign Types

- Product launches
- Brand awareness
- Seasonal campaigns
- Promotional campaigns
- Event marketing
- Content marketing

## Related

- `_brands/` - Brands being promoted
- `_products/` - Products being marketed
- `99-area/media/` - Campaign assets
