# Marketing Assets

This folder contains references and links to marketing assets stored in the media library.

## Asset Categories

### Brand Assets
- Logos and brand marks
- Brand guidelines
- Color palettes
- Typography guides

### Design Templates
- Social media templates
- Email templates
- Presentation templates
- Document templates

### Asset Libraries
- Stock photo collections
- Icon libraries
- Illustration sets
- Video footage

## Asset Storage

Physical assets are stored in:
- `99-area/media/images/` - Image files
- `99-area/media/videos/` - Video files
- `99-area/media/files/` - Other file types

This folder contains:
- Asset catalogs and indexes
- Usage guidelines
- Asset request forms
- Asset management documentation
