# Cross-Company Resources

This folder contains resources, policies, and procedures that apply across all business units in the RayCorp ecosystem.

## Structure

```
_cross-company/
├── faqs/           # Frequently Asked Questions (cross-company)
├── procedures/     # Standard Operating Procedures (SOPs)
├── policies/       # Company-wide policies
├── guidelines/     # Best practices and guidelines
└── templates/      # Document templates
```

## Purpose

Resources in this folder are:
- **Universal**: Apply to all business units (Lunaranger, Dianranger, Raycademy, Agroranger, Cosranger)
- **Standardized**: Ensure consistency across the organization
- **Centralized**: Single source of truth for cross-company information

## Usage

When creating unit-specific content, reference these cross-company resources to maintain consistency while adding unit-specific details.

## Related Folders

- `_fundamental/` - Core organizational structure and frameworks
- `_holding/` - Holding company (Rayandra Corporation) information
- `unit-bisnis/` - Individual business unit knowledge bases
