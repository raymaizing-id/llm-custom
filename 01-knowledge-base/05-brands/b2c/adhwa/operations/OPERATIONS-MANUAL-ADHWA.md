## 📋 ORDER FULFILLMENT

### Order Processing Flow

**Step 1: Order Received**
- System notification
- Order verification
- Payment confirmation
- Stock check

**Step 2: Picking**
- Pick list generation
- Product picking
- Quantity verification
- Batch number recording

**Step 3: Quality Check**
- Product condition check
- Expiry date check (min 6 months)
- Halal certification verification
- Packaging integrity

**Step 4: Packing**
- Bubble wrap for fragile items
- Adhwa branded packaging

**Step 5: Shipping**
- Courier handover

- Customer notification

## 🔄 RETURNS & EXCHANGES

### Return Policy

**Eligible Returns**:
- Damaged products
- Wrong items sent
- Expired products
- Defective products

**Return Window**: 7 days from receipt

**Process**:
1. Customer reports issue
2. Photo evidence required
3. Return approval
4. Product return (if needed)
5. Replacement/refund

### Exchange Process

**Eligible Exchanges**:
- Wrong shade/variant
- Unopened products
- Within 7 days

**Process**:
1. Exchange request
2. Product return
3. New product shipped
4. Free shipping both ways
