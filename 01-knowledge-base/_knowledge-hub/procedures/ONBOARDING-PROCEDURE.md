# 🎯 Onboarding Procedure
## New Employee Onboarding Process

> **Purpose**: Standardized onboarding untuk new employees  
> **Status**: Active  
> **Last Updated**: 2026-03-12

---

## 📋 PRE-ARRIVAL

### Before First Day
- [ ] Prepare workspace
- [ ] Set up computer
- [ ] Create accounts
- [ ] Prepare welcome package
- [ ] Notify team
- [ ] Schedule meetings

### Welcome Package
- Company handbook
- Org chart
- Key contacts
- Office map
- Parking info
- IT setup guide

---

## 🎪 FIRST DAY

### Morning (9:00 AM)
- [ ] Welcome & greeting
- [ ] Office tour
- [ ] IT setup
- [ ] Workspace setup
- [ ] Meet manager

### Afternoon (1:00 PM)
- [ ] Team introduction
- [ ] Department overview
- [ ] Role overview
- [ ] First tasks
- [ ] Lunch with team

---

## 📚 FIRST WEEK

### Day 1-2
- [ ] Complete paperwork
- [ ] IT training
- [ ] Security training
- [ ] Company overview
- [ ] Department overview

### Day 3-5
- [ ] Role-specific training
- [ ] Meet key stakeholders
- [ ] First projects
- [ ] Feedback session
- [ ] Weekly check-in

---

## 🎓 FIRST MONTH

### Week 2-3
- [ ] Deeper role training
- [ ] Shadowing
- [ ] First independent tasks
- [ ] Feedback
- [ ] Adjustment check-in

### Week 4
- [ ] 30-day review
- [ ] Performance feedback
- [ ] Goal setting
- [ ] Next steps
- [ ] Celebration

---

## 📊 ONBOARDING CHECKLIST

- [ ] Paperwork complete
- [ ] IT setup done
- [ ] Accounts created
- [ ] Training completed
- [ ] Team introduced
- [ ] Role understood
- [ ] First tasks assigned
- [ ] Manager check-in done
- [ ] 30-day review scheduled
