# ✅ Approval Procedure
## Standard Approval Process

> **Purpose**: Standardized approval process untuk semua requests  
> **Status**: Active  
> **Last Updated**: 2026-03-12

---

## 📋 APPROVAL LEVELS

### Level 1: Manager Approval
- Budget: <Rp 10M
- Timeline: <1 month
- Scope: Small changes
- Approver: Direct manager

### Level 2: Director Approval
- Budget: Rp 10M - Rp 100M
- Timeline: 1-3 months
- Scope: Medium changes
- Approver: Department director

### Level 3: Executive Approval
- Budget: >Rp 100M
- Timeline: >3 months
- Scope: Major changes
- Approver: C-level executive

---

## 🔄 APPROVAL PROCESS

### Step 1: Prepare Request
- [ ] Complete form
- [ ] Attach documentation
- [ ] Define business case
- [ ] Estimate budget
- [ ] Set timeline

### Step 2: Submit Request
- [ ] Submit to manager
- [ ] Provide context
- [ ] Answer questions
- [ ] Provide updates

### Step 3: Review
- [ ] Manager reviews
- [ ] Asks questions
- [ ] Requests changes
- [ ] Approves or rejects

### Step 4: Escalation (if needed)
- [ ] Submit to director
- [ ] Provide additional info
- [ ] Address concerns
- [ ] Get approval

### Step 5: Implementation
- [ ] Proceed with plan
- [ ] Track progress
- [ ] Report status
- [ ] Complete project

### Tracking
- Approval time
