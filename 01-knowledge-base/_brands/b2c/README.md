# B2C Brands (Business-to-Consumer)

This folder contains all consumer-facing brands across the RayCorp ecosystem.

## Brand Categories

### Cosmetics & Skincare (Lunaranger)
Consumer beauty and skincare brands

### Agriculture (Agroranger)
Direct-to-consumer agricultural products and organic offerings

### Education (Raycademy)
Consumer education programs and courses

### Consulting (Cosranger)
Individual consulting and coaching services

### Digital Services (Dianranger)
Consumer-facing digital products and services

## Brand Documentation Structure

Each brand should have:
```
brand-name/
├── README.md                  # Brand overview
├── brand-identity.md          # Visual identity, voice, values
├── products.md                # Product catalog
├── target-market.md           # Target audience
└── marketing-strategy.md      # Marketing approach
```

## Related Folders

- `_products/` - Products sold under these brands
- `_marketing/` - Marketing campaigns for B2C brands
- `unit-bisnis/` - Business units owning these brands
