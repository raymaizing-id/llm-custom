# B2B2C Brands (Business-to-Business-to-Consumer)

This folder contains hybrid business model brands where businesses sell to other businesses who then sell to consumers.

## Brand Categories

### White-Label Products (Lunaranger)
- Private label cosmetics
- White-label skincare
- Custom formulation services

### Distribution Partnerships (Multiple Units)
- Distributor programs
- Reseller partnerships
- Channel partner brands

### Franchise Models (Multiple Units)
- Franchise opportunities
- Licensed operations
- Partnership programs

### Platform Services (Dianranger)
- SaaS platforms for businesses serving consumers
- Marketplace platforms
- Technology licensing

## Brand Documentation Structure

Each brand should have:
```
brand-name/
├── README.md                  # Brand overview
├── partner-model.md           # Partnership structure
├── partner-requirements.md    # Partner criteria
├── support-resources.md       # Partner support
└── end-consumer-strategy.md   # Consumer-facing approach
```

## Related Folders

- `_products/` - Products/services in B2B2C model
- `_marketing/` - Marketing for both partners and consumers
- `unit-bisnis/` - Business units owning these brands
