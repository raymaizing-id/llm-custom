# PPIC (Production Planning & Inventory Control)

## Quick Facts
- **Type**: Manufacturing Department
- **Parent**: Lunaray Beauty Factory

## Department Overview

### Vision
Mengoptimalkan perencanaan produksi dan kontrol inventori untuk efisiensi maksimal.

### Mission
- Plan and schedule production efficiently
- Optimize inventory levels
- Ensure material availability
- Minimize production downtime
- Balance supply and demand

### Core Functions
1. **Production Planning** - Production scheduling
2. **Material Planning** - Material requirement planning (MRP)
3. **Inventory Control** - Stock level management
4. **Capacity Planning** - Production capacity optimization
5. **Work Order Management** - Production order coordination

## Team Structure

### Production Planner (2 people)
### Material Requirement Planner (1-2 people)
### Material Control Officer (1 person)
### Packaging Planner (1 person)
### Monitoring Scheduler (1 person)
### Work Order Admin (1 person)

## Key Responsibilities

### 1. Production Planning
- **Master Production Schedule (MPS)**: Monthly/weekly production plan
- **Capacity Planning**: Ensure adequate production capacity
- **Line Scheduling**: Assign products to production lines
- **Priority Management**: Prioritize urgent orders
- **Schedule Optimization**: Minimize changeovers and downtime

### 2. Material Planning
- **Material Requirement Planning (MRP)**: Calculate material needs
- **Purchase Requisitions**: Generate PRs for procurement
- **Material Availability**: Ensure materials ready for production
- **Lead Time Management**: Track supplier lead times
- **Safety Stock**: Maintain appropriate safety stock levels

### 3. Inventory Control
- **Stock Monitoring**: Track inventory levels
- **Reorder Points**: Set and monitor reorder points
- **Stock Analysis**: ABC analysis, slow-moving analysis
- **Inventory Optimization**: Minimize excess and obsolete inventory
- **Cycle Counting**: Regular inventory verification

### 4. Work Order Management
- **Work Order Creation**: Generate production work orders
- **Work Order Release**: Release to production
- **Progress Tracking**: Monitor production progress
- **Work Order Closure**: Close completed work orders
- **Documentation**: Maintain work order records

## Planning Process

### Monthly Planning Cycle
1. **Sales Forecast** (Day 1-5): Receive sales forecast from Marketing
2. **Capacity Check** (Day 6-10): Verify production capacity
3. **Material Planning** (Day 11-15): Calculate material requirements
4. **Schedule Creation** (Day 16-20): Create production schedule
5. **Schedule Release** (Day 21-25): Release to production
6. **Monitoring** (Day 26-30): Monitor and adjust as needed

### Weekly Planning
- Review weekly production targets
- Adjust schedule based on actual progress
- Coordinate with Supply Chain for material availability
- Communicate schedule changes to Production

### Daily Planning
- Monitor daily production output
- Address urgent orders or issues
- Coordinate material delivery to production lines
- Update production status
